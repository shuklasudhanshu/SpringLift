## ğŸš€ How to Use

### 1. Quick Start (5 minutes)

```bash
# Install dependencies
pip install -r requirements.txt

# Create configuration
cp .env.example .env

# Edit .env with your API keys (optional)
# OPENAI_API_KEY=sk-xxx
# ANTHROPIC_API_KEY=sk-ant-xxx

# Start server
uvicorn springlift.main:app --reload
```

### 2. Scan a Java Project

```bash
curl -X POST "http://localhost:8000/scan" \
  -H "Content-Type: application/json" \
  -d '{
    "project_path": "/absolute/path/to/your/java/project",
    "use_ai": true,
    "ai_provider": "openai"
  }'
```

### 3. Review Results

The response includes:
- **Scan ID** for future retrieval
- **Output Path** to modernized project: `{project}_modernized/`
- **File-by-file Analysis**: Issues, suggestions, transformations
- **Dependency Recommendations**: Maven/Gradle upgrades
- **Statistics**: Files analyzed, issues found, transformations applied
